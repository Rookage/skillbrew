SkillBREW

技能酿造局：丢链接进去，自动消化、自动评估、自动去重，真实安装后留下透明台账，还能在 Claude Code 与 Codex CLI 之间双运行时中立运行。让你清楚 Agent 里有什么、怎么调。

GitHub 仓库 快速开始

六大核心能力

🧠

自动去重

把本地已装能力、历史台账和新推荐三者交叉比对。名字命中直接跳过，描述高度重叠标为整并候选，其余才算新增。

⭐

自动评估

结合一手溯源（仓库存在性、星数、维护状态）和本机能力画像，给出「值得装 / 挑着装 / 整源跳过」的结构化建议。

⚡

真实安装

默认 dry-run，授权后才真落盘。支持 Skill、MCP、代码、配置、Prompt 多种形态，按运行时约定落到正确位置。

📊

透明台账

装完自动生成 RECORD.md 台账与 DASHBOARD.md 看板，记录来源、时间、调用方式与就绪状态，越装越清楚。

🤖

通用安装器

AI 自动读源头仓库推断安装方法，试跑验证后装，缺什么就问你补什么。缓存命中秒装，catalog 种子兜底，装不了就明白告诉你卡在哪。

🖥️

双运行时中立

自动识别 Claude Code 与 Codex CLI，自动切换 Skill 目录、MCP 配置和克隆缓存路径，可用环境变量逐项覆盖。

8 步管线

从丢链接到能力落地，每一步都可独立运行、可审计、可回滚。

采集→ 理解→ 消化→ 溯源→ 判断→ 去重→ 安装→ 记录

中间产物全部落在 data/sources/<id>/，随时可复查。

快速开始

# 1. 克隆仓库
git clone https://github.com/Rookage/skillbrew.git
cd skillbrew

# 2. 复制配置并填入你自己的 key
cp .env.example .env
#   编辑 .env：TEXT_API_KEY / VISION_API_KEY（推荐 NVIDIA NIM，免费快 50x）

# 3. 安装（可编辑，注册 skillbrew 命令）
pip install -e .

# 4. 丢一个链接进去（以抖音 MCP 推荐视频为例）
skillbrew run https://v.douyin.com/LcTwRFgJqas/

# 5. 溯源 → 评估 → 去重
skillbrew verify data/sources/douyin_7650401218518387995/
skillbrew recommend data/sources/douyin_7650401218518387995/
skillbrew dedup data/sources/douyin_7650401218518387995/

# 6. 先看计划，确认后再真装
skillbrew install data/sources/douyin_7650401218518387995/
skillbrew install data/sources/douyin_7650401218518387995/ --approve

# 6b. 启用 AI 推断（catalog 未收录的 MCP 也能自动装）
skillbrew install data/sources/douyin_7650401218518387995/ --approve --ai-infer

# 7. 生成台账 + 看板
skillbrew record data/sources/douyin_7650401218518387995/

视频获取与语音转文字依赖较大，可按需安装：pip install -e ".[media]"。
